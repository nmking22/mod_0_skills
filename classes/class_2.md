## Restaurant Class 2

**Class:** `FineDining`

**Attributes:**

`number_of_chefs` (integer)

`owner_name` (string)

`is_open` (boolean)

`inventory` (array)

**Methods:**

`fire_chef` (decreases number_of_chefs)

`sell_restaurant` (changes owner_name)

`close_restaurant` (changes is_open from true to false)

`accept_food_order` (adds strings to inventory array)
